# Requirement Index

## Phase mapping

- **Phase 6A — Clients CRM foundation**
  - **REQ-011**: Admin clients list foundation exists and is role-gated.
  - **REQ-012**: Admin clients search foundation supports name/email/phone query.
  - **REQ-013**: Admin clients filter foundation supports all/upcoming/no-upcoming/inactive.
  - **REQ-014**: Client detail panel foundation with appointment context summary and editable fields.
  - **REQ-015**: Add client foundation enforces first name + last name + one contact path.
  - **REQ-016**: Edit client foundation supports non-destructive updates, including notes.
  - **REQ-017**: CRM empty and not-found states exist for list/detail flows.
