# Requirement Index

## Public Scheduler (Phase 3A Scope)
- REQ-001: Public scheduling page foundation route and responsive baseline.
- REQ-002: Service selection flow on public scheduler.
- REQ-003: Direct-link behavior for appointment type routes.
- REQ-004: Date/time entry and slot selection foundation.
- REQ-005: Multi-time selection mode foundation.
- REQ-006: Recurring booking modal foundation.
- REQ-007: Public visibility filtering (exclude private catalog items unless direct-link, exclude admin-only add-ons, exclude internal-use forms).

## Deferred Beyond Phase 3A
- REQ-008+: Payment flows, client self-service, confirmation self-service, and admin surfaces are deferred.
