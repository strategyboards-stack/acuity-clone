- **Phase 1 — Platform shell**
  - **REQ-001**: Normalize monorepo into apps/packages structure with shared configuration.
  - **REQ-002**: Provide web/app, api/app, worker/app executable shells.
  - **REQ-003**: Implement auth shell plus separate workspace and account host-shell route groups.
  - **REQ-004**: Implement entitlement and feature-gating middleware using role/plan/trial/dependency/verification inputs.
  - **REQ-005**: Keep Manage Users and Billing as host-level account surfaces, not scheduling tabs.
  - **REQ-006**: Add foundational contracts, database schema, testing package, and phase docs updates.

- **Deferred for later phases**
