# Requirement Index

## Phase 3A Validation Scope

- **REQ-P3A-BOOT-01**: Repository must bootstrap in Codex with pinned package manager behavior (`pnpm@10.13.1`) and no manual intervention.
- **REQ-P3A-BOOT-02**: `pnpm install` must complete successfully for the current Phase 3A repository state.
- **REQ-P3A-BOOT-03**: Root `pnpm typecheck`, `pnpm test`, and `pnpm build` must pass.
- **REQ-P3A-DOM-01**: Public scheduler foundation marker remains explicitly Phase 3A and excludes Phase 3B behavior.
- **REQ-P3A-DOC-01**: Derived reporting artifacts must be present and updated after validation.
