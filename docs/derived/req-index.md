# Requirement Index

## Phase 2B — Catalog and Forms Configuration

- **REQ-2B-AT**: Admin configuration for appointment types, including private type controls and direct-link metadata.
- **REQ-2B-AO**: Admin configuration for add-ons, including admin-only visibility controls.
- **REQ-2B-CP**: Admin configuration for appointment coupons with blank expiration interpreted as non-expiring.
- **REQ-2B-IF**: Admin configuration for intake forms, including internal-use flag not exposed in public surfaces.
- **REQ-2B-GATE**: Server-aware feature gating for configuration actions based on role, plan, trial, dependency, verification state.
- **REQ-2B-PUBSEP**: Public catalog projection must exclude private appointment types, admin-only add-ons, and internal-use forms.
