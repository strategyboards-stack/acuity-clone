# Requirement Index

## Phase 10A — Foundation Stabilization

- **REQ-P10A-001**: Repository bootstraps with `pnpm` in Codex without manual intervention.
- **REQ-P10A-002**: `pnpm install` completes successfully.
- **REQ-P10A-003**: Monorepo minimum structure exists (`apps/*`, `packages/*`, `docs/*`).
- **REQ-P10A-004**: Validation commands pass: `pnpm typecheck`, `pnpm test`, `pnpm build`.
- **REQ-P10A-005**: Session reporting artifacts are updated after validation.
