# Requirement Index

## Phase 1 Scope (Stabilization only)

- **REQ-P1-BOOT-01**: Repository bootstrap must be installable in Codex without manual corepack intervention.
- **REQ-P1-BOOT-02**: Workspace must expose stable root scripts for `typecheck`, `test`, and `build`.
- **REQ-P1-BOOT-03**: Minimal package boundaries must compile and validate in CI-style commands.
- **REQ-P1-DOC-01**: Derived reporting artifacts must be present and updated after validation.

## Out of Scope for this run

- Any Phase 2 feature implementation.
- Product-surface behavior changes beyond bootstrap stabilization.
